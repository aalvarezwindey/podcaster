# Podcaster

This app let you browse some of the top Podcasts available in iTunes.

## Development

### Install dependencies

```bash
npm run install
```

### Run in dev

```bash
npm run start
```

### Run tests

```bash
npm run test
```

### Run in production mode

```
npm run start:production
```
